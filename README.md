## Get started

1. Copy this repository

```bash
git clone https://github.com/jotadevelopers/uno-admin.git
```
2. Access the created directory 

```bash
cd uno-admin
```

3. Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

4. Start the development server on http://localhost:3000

```bash
npm run dev
```
